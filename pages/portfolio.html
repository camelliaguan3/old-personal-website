tank game


Start Date: jul 2022

End Date: aug 2022


Description:

Tank Game is the very first game that I coded from scratch. By implementing a game such as this, I learned more about the process of game development and creating interfaces. I was be able to design a game, implement it, and test the game throughout my process. The game uses Turtle module to add more flexibility and simplicity to the game that the Kivy module, which I originally tried utilizing, could not provide.


Languages/Frameworks: Python


project

playful plants


Start Date: apr 2022

End Date: may 2022


Description:

Playful Plants is a project designed for the Playful Plants Project, allowing users to browse, sort, and filter plants. Administrators of Playful Plants can also access the Administrator page, allowing them to edit plant entries that can then be viewed by consumers. For this project, I documented a design journey to identify audience goals, wireframe website designs, and plan out a database and formulate SQL queries.


Languages/Frameworks: HTML, CSS, JavaScript, JQuery, PHP, SQLite, Docker

Description:

Hometown Myth - Choose Your Own Adventure is a project created for CLASS 2604: Greek Myth in order to showcase a familiar myth. This project makes use of Javascript and JQuery to recreate the myth of Osiris and his rebirth as the God of the Dead and incorporates interactive elements that allow a user to navigate through a choose your own adventure game.
